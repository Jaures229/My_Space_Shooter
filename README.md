# My_Space_Shooter

## Description
"My_Space_Shooter" est un jeu de tir spatial développé en utilisant la bibliothèque SFML2. Ce projet a pour objectif de fournir une expérience de jeu immersive et divertissante dans l'univers spatial.

## Prérequis
Avant de compiler le projet, assurez-vous d'avoir installé les éléments suivants :

- CMake (version 3.10 ou supérieure)
- Make (pour les systèmes basés sur Unix)
- SFML2 (Simple and Fast Multimedia Library)

## Compilation

### Avec CMake

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Jaures229/My_Space_Shooter.git
   cd My_Space_Shooter
   ```

2. Créez un répertoire de construction :
   ```bash
   mkdir build
   cd build
   ```

3. Exécutez CMake pour configurer le projet :
   ```bash
   cmake ..
   ```

4. Compilez le projet :
   ```bash
   make
   ```

### Avec Makefile

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Jaures229/My_Space_Shooter.git
   cd My_Space_Shooter
   ```

2. Compilez le projet directement avec Make :
   ```bash
   make
   ```

## Lancer le jeu
Après la compilation, vous pouvez lancer le jeu avec la commande suivante :
```bash
./My_Space_Shooter
```
